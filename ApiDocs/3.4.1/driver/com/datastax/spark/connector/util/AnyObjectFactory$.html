  14.  def isRealMemberClass[T](clazz: Class[T]): Boolean

    This method checks if the class is a member class which requires providing a reference to the enclosing in its constructors.

    This method checks if the class is a member class which requires providing a reference to the enclosing in its constructors. We cannot just check it by invoking java.lang.Class#isMemberClass because it will return true for classes enclosed in Scala objects. They do not accept reference to enclosing class in their constructors, and therefore they need to be treated as normal, top level classes.
